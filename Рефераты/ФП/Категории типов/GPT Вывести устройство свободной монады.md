

Начальные условия:
1. Эндофунктор f: (𝒞)→(𝒞). Это описание наших "базовых операций" или "эффектов".
2. Категория f-Алгебр (f-Alg):
  •  Объекты: пара (A, α), где A — объект из (𝒞), а α: f(A) → A — морфизм (описывает, как эффекты преобразуются в значения A).
  •  Морфизмы: h: (A, α) → (B, β) — морфизм h: A → B из (𝒞), удовлетворяющий условию гомоморфизма алгебр: h ∘α = β∘ f(h).
3. Забывающий функтор U: f-Alg →(𝒞): U(A, α) = A. Он "забывает" структуру α.

—

 Шаг 1: Формулировка задачи поиска свободного функтора

Мы хотим построить свободный функтор F: (𝒞)→ f-Alg, который является левым адъюнктом к U. Это означает, что для каждого объекта x ∈(𝒞) нам нужно найти "наименьшую" или "самую свободную" f-алгебру, которая "содержит" x.

По определению левой адъюнкции, для каждого объекта x ∈(𝒞) существует свободная f-алгебра F(x) = (M_x, ϕ_x) (где M_x — её носитель, ϕ_x: f(M_x) → M_x — её структура) и морфизм η_x: x → M_x (называемый единицей сопряжения).

Эти объекты и морфизмы должны удовлетворять универсальному свойству:
Для ×любой× f-алгебры (A, α) и ×любого× морфизма g: x → A в (𝒞), существует единственный гомоморфизм h: (M_x, ϕ_x) → (A, α) такой, что h ∘η_x = g.

 Шаг 2: Вывод структуры носителя M_x и морфизмов η_x, ϕ_x

Давайте подумаем, как должен быть устроен M_x, чтобы удовлетворить универсальному свойству.

1. Начало: Поскольку существует морфизм η_x: x → M_x, то M_x должен быть достаточно большим, чтобы "содержать" все элементы из x. Это наше стартовое требование.

2. Эффекты: M_x является носителем f-алгебры, то есть у него есть морфизм ϕ_x: f(M_x) → M_x. Это значит, что если мы применим функтор f к элементам M_x, результаты этих "эффектов" также должны быть "содержаться" в M_x.

3. Гомоморфизм h: Морфизм h: M_x → A должен быть гомоморфизмом алгебр, то есть h ∘ϕ_x = α∘ f(h). Это ключевое условие.

Теперь мы строим M_x индуктивно (рекурсивно), основываясь на этих требованиях.

•  Базовый случай (глубина 0): Чтобы удовлетворить h ∘η_x = g, каждый элемент из x должен быть отображён в M_x через η_x. И эти элементы должны быть доступны для h. Пусть мы имеем Pure конструктор.
  M_x содержит все η_x(v) для v ∈ x
  И для них h(η_x(v)) = g(v).

•  Рекурсивный случай (глубина k+1): Теперь рассмотрим, что такое ϕ_x: f(M_x) → M_x. Если e ∈ f(M_x), то ϕ_x(e) — это элемент M_x. И для него должно выполняться условие гомоморфизма.
  h(ϕ_x(e)) = α(f(h)(e))
  Это означает, что любой "эффект" e над элементами M_x (которые сами могут быть результатами предыдущих эффектов) может быть "свёрнут" функцией ϕ_x обратно в M_x.
  Пусть мы имеем Free конструктор.

Это приводит нас к мысли, что M_x должен быть построен как объединение (сумма) двух видов элементов:
1. Те, что пришли непосредственно из x (через η_x).
2. Те, что являются результатом применения f к уже существующим элементам M_x (через ϕ_x).

Поэтому, структура M_x должна удовлетворять изоморфизму:
M_x ≅ x + f(M_x)
(где + обозначает дизъюнктное объединение/сумму объектов).

Определим конструкторы:
•  η_x: x → M_x — это просто вложение в первое слагаемое суммы: inL: x → x + f(M_x).
•  ϕ_x: f(M_x) → M_x — это просто вложение во второе слагаемое суммы: inR: f(M_x) → x + f(M_x).

Теперь мы должны проверить, что такая конструкция M_x с этими η_x, ϕ_x удовлетворяет универсальному свойству.

Проверка универсального свойства:
Даны (A, α) и g: x → A. Мы ищем h: M_x → A такой, что h ∘η_x = g и h ∘ϕ_x = α∘ f(h).
Поскольку M_x ≅ x + f(M_x), мы можем определить h рекурсивно, как единственный морфизм из начальной алгебры:
1. Для элементов, пришедших из x (т.е. inL(v)): h(inL(v)) = g(v). Это удовлетворяет h ∘η_x = g.
2. Для элементов, пришедших из f(M_x) (т.е. inR(e)): h(inR(e)) = α(f(h)(e)). Это удовлетворяет h ∘ϕ_x = α∘ f(h), поскольку e ∈ f(M_x) и f(h) корректно отображает f(M_x) в f(A).

Такое определение h является единственным, потому что каждый элемент M_x либо является элементом из x, либо получен применением f к предыдущим элементам. Базовый случай определяется g, рекурсивный — α. Нет других способов построения M_x, поэтому нет других способов определения h.

 Шаг 3: Вывод устройства свободной монады как неподвижной точки

Мы только что конструктивно вывели, что носитель M_x свободной f-алгебры для объекта x должен быть изоморфен:
M_x ≅ x + f(M_x)

Эта форма Z ≅ x + f(Z) — это в точности определение неподвижной точки функтора G_x: (𝒞)→(𝒞), который определяется как G_x(Z) = x + f(Z).

Таким образом, мы вывели, что свободная монада (а точнее, её носитель для каждого x) является наименьшей неподвижной точкой функтора G_x. Если бы мы взяли большую неподвижную точку, она бы не была "свободной", так как имела бы избыточные структуры, не диктуемые f или x, и не удовлетворяла бы универсальному свойству единственности.

1. Начали с определения свободного функтора F ⊣ U через его универсальное свойство.
2. Проанализировали требования к носителю M_x и его структуре ϕ_x для удовлетворения этого универсального свойства.
3. На основании этих требований вывели, что M_x должен быть изоморфен x + f(M_x).
4. Признали в этой форме определение неподвижной точки функтора Z ↦ x + f(Z).
